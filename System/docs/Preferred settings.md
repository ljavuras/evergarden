# Preferred settings

## Options

### Editor

#### Display

| Setting                | Value  | Description                                                            |
| ---------------------- | ------ | ---------------------------------------------------------------------- |
| Properties in document | Hidden | [[Properties#Properties are hidden away\|Properties are hidden away]]. |
### Appearance

#### Interface

| Settings           | Value | Description                                                                                                                                     |
| ------------------ | ----- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Show inline title  | false | Title of the note is already visible, because [[Headings should form tables of contents]]. Also gives the option to put things above the title. |
| Show tab title bar | false | Full path is irrelevant in most cases. Prefer `Rename file` command (`F2`) over clicking on title bar.                                          |
| Show ribbon        | false | Prefer keyboard over mouse, and reduces visual clutter.                                                                                         |
